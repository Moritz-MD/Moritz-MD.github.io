Posting convention

For each new post, create a file at blog/posts/YYYY-MM-DD-title.md (or .html while plain HTML is used).

Each post should include visible metadata fields:

  • Title: human-readable post title
  • Date: publication date in YYYY-MM-DD
  • Tags: topic tags like methods, theory, experiments

When using plain HTML, manually add each new entry to this index. If migrated to Jekyll later, keep the same directory and URL pattern.
